# Portfolio Projects - Real Code, Real Skills

This portfolio contains **8 comprehensive projects** demonstrating full-stack development capabilities using HTML, CSS, JavaScript, Bootstrap, APIs, and modern web development practices.

## 🚀 Projects Overview

### ✅ 1. Ultimate Landing Page
**Location**: `/landing-page/`
- **Features**: Complete layout with Navbar, Hero, Categories, Features, Streaming/Media sections, Testimonials, CTA, and Footer
- **Tech Stack**: HTML5, CSS3, Bootstrap 5, JavaScript
- **Highlights**: Smooth animations, responsive design, interactive elements

### ✅ 2. Master Form Suite  
**Location**: `/form-suite/`
- **Features**: Multi-step forms including Get-Matched, Contact, Auth (Login/Signup), Newsletter with comprehensive JS validation
- **Tech Stack**: HTML5, CSS3, JavaScript, Form Validation
- **Highlights**: Progress indicators, real-time validation, user-friendly error handling

### ✅ 3. Advanced Search Filters
**Location**: `/search-filters/`
- **Features**: Real-time filtering logic with multiple parameters (price, category, location) and sorting
- **Tech Stack**: JavaScript, HTML5, CSS3, Dynamic DOM Manipulation
- **Highlights**: Instant search results, category filtering, price range slider

### ✅ 4. AI-Style ChatBot
**Location**: `/chatbot/`
- **Features**: Interactive chat interface with keyword-based auto-responses and dynamic DOM updates
- **Tech Stack**: JavaScript, HTML5, CSS3, WebSocket Simulation
- **Highlights**: Typing indicators, emoji support, conversation management

### ✅ 5. E-commerce Logic
**Location**: `/ecommerce/`
- **Features**: Product Cart system with state management, Order Summary calculation, and Checkout page
- **Tech Stack**: JavaScript, HTML5, CSS3, Local Storage
- **Highlights**: Shopping cart, checkout process, wishlist functionality

### ✅ 6. User Dashboard
**Location**: `/user-dashboard/`
- **Features**: Protected area for users to manage profiles, change passwords, and view saved activity
- **Tech Stack**: JavaScript, HTML5, CSS3, Authentication Simulation
- **Highlights**: Profile management, activity tracking, theme switching

### ✅ 7. Simple CMS
**Location**: `/cms/`
- **Features**: Admin panel to Create, Read, Update, and Delete content (CRUD) for the landing page
- **Tech Stack**: JavaScript, HTML5, CSS3, Local Storage CRUD Operations
- **Highlights**: Drag-and-drop reordering, media management, preview functionality

### ✅ 8. Live Data Dashboard
**Location**: `/live-dashboard/`
- **Features**: Fetch real-time data from public APIs (Weather/Crypto) and visualize it in cards
- **Tech Stack**: JavaScript, APIs, HTML5, CSS3, Real-time Updates, Chart.js
- **Highlights**: Auto-refresh, data visualization, export functionality

## 🛠 Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+), Bootstrap 5
- **Data Storage**: Local Storage (for demo purposes)
- **APIs**: Public APIs (Weather, Cryptocurrency), Chart.js
- **Tools**: Modern JavaScript, Responsive Design, DOM Manipulation

## 🚀 Getting Started

Each project is self-contained and can be run independently. Simply open the `index.html` file in any modern web browser.

### Quick Start
```bash
# Navigate to any project directory
cd landing-page  # or form-suite, search-filters, etc.

# Open index.html in your browser
open index.html
```
